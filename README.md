# Notes

A minimal note-taking website written in **Typst**, compiled to PDF for browser-native viewing.

## Features

- **Write in Typst** — expressive markup language
- **Compiled to PDF** — `typst compile` produces PDF with fully selectable text
- **Instant loading** — browser-native PDF viewer, no WASM or plugins
- **File management** — sidebar with folder tree
- **Dark/Light mode** — theme toggle with system preference detection
- **GitHub Pages** — push to `main`, auto-deploys via CI

## Quick Start

### 1. Fork / Clone

```bash
git clone https://github.com/YOUR_USERNAME/note_web.git
cd note_web
```

### 2. Add your notes

Create `.typ` files in `notes/`:

```bash
echo '= My First Note

Hello, world!' > notes/my-first-note.typ
```

### 3. Enable GitHub Pages

1. **Settings → Pages → Source: "GitHub Actions"**
2. Push to `main` — CI compiles and deploys

Site live at `https://YOUR_USERNAME.github.io/note_web/`

## Local Development

```bash
# Install typst & poppler-utils (for pdfinfo)
# Build
node scripts/build.js

# Serve
python3 -m http.server 8080 --bind 127.0.0.1
```

## Writing Notes

Notes are plain Typst files. See [Typst docs](https://typst.app/docs/) for syntax.

## How It Works

1. **CI build:** `typst compile` converts each `.typ` to PDF
2. **Browser:** PDFs embedded via `<embed>`, rendered natively with text selection
3. A Node script generates `manifest.json` with page counts via `pdfinfo`
4. Entirely static — no server, no runtime dependencies
